# Candle Matrix

Умное устройство атмосферной подсветки на базе светодиодной матрицы 9×16, имитирующее мерцание пламени свечи. Построено на микроконтроллере **ESP32-S3** с поддержкой Wi-Fi и встроенным веб-интерфейсом.

## Возможности

- **Анимация свечи** — плавный эффект мерцания света на светодиодной матрице 9×16
- **Автоматический режим по солнцу** — устройство автоматически включается в сумерках и выключается на рассвете по GPS-координатам; время восхода/заката и сумерек вычисляется на устройстве по формулам NOAA (без внешнего API)
- **Отображение фазы Луны** — отдельный режим анимации фазы Луны
- **Визуализация положения Солнца** — режим индикатора текущего положения Солнца в реальном времени
- **Регулируемая яркость** — 0–100% через веб-интерфейс или HTTP API
- **Синхронизация времени через NTP** — точное локальное время с настраиваемыми NTP-сервером и POSIX-таймзоной
- **Веб-интерфейс** — адаптивный UI, раздаваемый из LittleFS (без зависимости от облака)
- **HTTP REST API** — программное управление и мониторинг устройства
- **Метрики Prometheus** — endpoint `/metrics` для интеграции с дашбордами Grafana
- **Captive portal** — простая настройка Wi-Fi при первом запуске
- **mDNS** — доступ по имени хоста (например, `http://candle.local`)

## Программный стек

- **Framework:** Arduino (через PlatformIO)
- **Веб-сервер:** ESPAsyncWebServer
- **Файловая система:** LittleFS
- **JSON:** ArduinoJson 7
- **LED-драйвер:** Adafruit IS31FL3731 Library
